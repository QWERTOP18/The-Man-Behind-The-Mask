LIBRARY
     Standard C Library (libc, -lc)

SYNOPSIS
     #include <string.h>

     void *
     XXXXXX(void *b, int c, size_t len);

DESCRIPTION
     The XXXXXX() function writes len bytes of value c (converted to an
     unsigned char) to the string b.

RETURN VALUES
     The XXXXXX() function returns its first argument.

SEE ALSO
     bzero(3), memset_pattern(3), memset_s.3, swab(3), wmemset(3)

STANDARDS
     The XXXXXX() function conforms to ISO/IEC 9899:1990 (“ISO C90”).
